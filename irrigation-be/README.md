# Smart Irrigation System
A Spring Boot application that monitors soil moisture and automates garden irrigation using connected devices (soil moisture sensor and electric valves)
.
## Getting Started

### Prerequisites
 Java 17+ 
 Maven 
 Docker (for db)

### Start (local H2)
 ./mvnw spring-boot:run
